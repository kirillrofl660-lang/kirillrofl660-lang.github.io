# Династия — лендинг студии создания сайтов

Современный одностраничный лендинг для веб-студии: приём заказов и предоставление ресурсов для запуска проектов.

## Стек

- **Vite + React 19 + TypeScript**
- **Chakra UI v3** — UI-компоненты и кастомная тема
- **React Spring** (`@react-spring/web`) — анимации появления, плавающие элементы, счётчики
- **lucide-react** — иконки

## Палитра бренда

| Цвет | HEX | Назначение |
| --- | --- | --- |
| Графит | `#363233` | тёмные секции, текст |
| Тёмная бронза | `#613708` | акценты, градиенты |
| Бронза | `#895C20` | основной акцент (кнопки) |
| Таупе | `#A38976` | вторичные элементы |
| Песочный | `#CABCAC` | фон, светлые блоки |

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # сборка в dist/
npm run preview  # предпросмотр сборки
```

## Структура

```
src/
  theme.ts              # тема Chakra с палитрой бренда
  main.tsx              # точка входа + ChakraProvider
  App.tsx               # сборка секций
  components/
    motion.tsx          # анимационные утилиты на React Spring
    Navbar.tsx          # шапка
    Hero.tsx            # первый экран
    Services.tsx        # услуги
    Process.tsx         # этапы работы
    Resources.tsx       # предоставляемые ресурсы
    Cases.tsx           # кейсы
    Pricing.tsx         # тарифы
    OrderCTA.tsx        # форма заявки
    Footer.tsx          # подвал
```

Форма заявки сейчас работает на клиенте (показывает подтверждение). Для приёма заказов
подключите backend/почтовый сервис в `components/OrderCTA.tsx` (обработчик `handleSubmit`).
